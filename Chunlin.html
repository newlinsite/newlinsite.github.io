Chunlin是建築建材經銷商與生產商，此專案是Chunlin為增加自家生產工廠品牌知名度、建立綠建材媒體，以及做為未來的東協生產設備的形與聯絡銷售網站。 公司產品自產自銷在傳統行銷上有豐富成果但在數位行銷方面卻沒有相關經驗，所以希望能在透過這次架站過程中建立公司的數位行銷方面的流程與方法，讓網站並不在是線上的靜態DM，並套用公司的其他營運公司上。

網站目標需求／目的：

1.建立Chunlin形象
2.東協市場設備服務的平台網站
　i.販售XXX生產設備
　ii.XXX生產技術轉移輔導
　iii.品牌與通路服務
　iv.設計服務
3.導流至鑽泥板網站 / 增加鑽泥板業務

整理目前有參考價的GA數據

在最初期我們在確定完網站目標與行程表後，希望能更進一步了解公司的目標族群，所以希望能在公司原本的其他網站上拿取資料，但GA基本設定與維護經營方面狀況皆不在正軌上，導致許說數據可能都是失真的數字，所以在切出更多自訂區隔與公司之前的行為做比對以利更了解網站流量來源與構成的使用者，右圖為某一流向來源與上架文章的關係（內容部分修改過）。

我們根據公司業務經驗與GA數據
經過討論後 將使用者分為族群Ｂ與族群Ｐ兩大類：

再分別針對族群Ｂ族群Ｐ問問自己兩個問題，：

「如何讓一位正在想要採購的買主找到，並留住他？」　－族群Ｂ

「如何讓正確的人接觸我們，並讓他印象深刻？」　－族群Ｐ

根據兩個問題，如何開始，我們最後將執行方式分成３個方向：

1.不只是在賣產品,讓使用者覺得與自己有關 - 實用的內容(內容行銷)

  1. 目的在於讓網站給人不只是在賣產品的感覺，除了公司產品也提供一些相關資 訊內容，如推廣綠建材、台灣森林相關資訊.

  2. Cooler系列的形象包裝，對於未來的綠色循環產品包裝

  3. 清楚的產品資訊與文案／產品規格圖像化等等：更容易辨識／吸引人的內容

  4. 部落格／分享／統整相關內容
    使用medium作為部落格載體
    medium與其他部落個相比：畫面簡潔、內容用品質較高、多深度文章
    使用medium優點：增加外部連結、medium身為部落格平台的產生的流量
    使用medium／外部部落格缺點：無法置入程式碼，給行銷用的數據掌控度較低

2.讓使用者更容易完成自己操作與搜尋：網站用戶體驗UX－這邊是後續Wireframe與UI討論的重點

  1. 對目標族群優化網站體驗，針對不同族群的入口頁面與使用流程量身打造

  2. 找出SEO關鍵字，並根據時間推移更新內容，使使用者能快速找到網站（比如：想找吸音建材，可以找到我們）

  3. 迅速找到想要的資訊，不迷航 (針對目的性強烈的客戶)

  4. 令人印象深刻的中菱形象，作出品牌的視覺區隔或是內容區隔

3.多樣、目標正確的導入方式（陌生開發）

  1. 建立信任感，到處都可以搜尋到我們，

  2. 配合社群行銷（部落格、YOUTUBE等等）

  3. 實體：名片上的Qrcord、DM上的Qrcord等等，或是其他東西，或是吸睛的宣傳物

  4. 平台：設計師＋、未來的東協綠建材平台、黃頁等等

User journey map

透過原業務統計數據，定義使用者與網站後續執行方向後，我們嘗試使用User journey map去描繪兩種使用者在使用網站時，所有可能的會經歷的過程與目的，並探討現階段可預期的錯誤並修正，以利下階段網站內容架構的產出。 此為簡化版的User jouney map，主為決定各時間點我們要給使用者什麼樣的資訊與引導的設計假設，上線後會再透過ＧＡ收集來的數據修正頁面、關鍵字、甚至於使用者的定義等等各項假設，持續優化各項網站體驗。

Chunlin web struciture

透過上面幾個階段，我們大概確定了使用者流程、網站目的等資訊後，接著我與主管開始討論個網站期待放置的內容，並使用依頁面為大單位，內容session為小單位，黃色為網站上的重點按鈕位置的頁面，結構方式來進行詳細的內容討論，討論面包括：是否能引起讀者興趣、對SEO的影響、頁面觀看流程是否順暢、能否順利導向轉換頁面等等。 注：此圖不是最後版本且內容有經過修改。



Wireframe

官方網站上並沒有太複雜的互動方式，主要功能方面只有部落格，我們依照web struciturｅ訂好的內容將其細節填滿並修改。
不過實際在wireframe填內容時發現網站內容困乏，許多頁面在整理時發現不知道如何填寫內容照片、文案缺少，目前公司也比較難以生出某些內容，導致修該了很多頁面和原本web struciturｅ時討論的內容。傳產可利用的元素過少，加上時間緊迫，導致Wireframe討論過程花很長時間，這次得到了很好的經驗下次在前期會更重視內容產生的可行性。
注：Wireframe討論過程此只兩張做為代表，不是最後版本且內容有經過修改。
