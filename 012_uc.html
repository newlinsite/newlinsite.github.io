Streaming Live HD

2018 - 2020

研究協助 / 應用情境定義 / UI Flow / GUI設計 / Library建立 / 銷售應用設計

多功一體的串流媒體工作站產品系列的APP設計
透過測試與協助銷售應用設計接觸用戶，持續優化產品

透過使用者訪談結果定義使用情境，規劃系列產品新功能開發、優化與情境設定、並建立應用內容協助業務銷售。

定義包含APP介面互動、硬體連接、網路狀態等等，並統一風格與APP使用行為，已達最好的使用體驗。與RD合作上使用Overflow與Zeplin協作、並建立Library協助RD減少開發時間，並讓讓系列產品更有一致性。 此產品榮獲2019年Good Design。
