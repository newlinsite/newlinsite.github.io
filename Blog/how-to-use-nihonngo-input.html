Windows 安裝「日本語」輸入法

以下是 Windows 安裝「日本語」輸入法的安裝步驟

1點選視窗右下角的語言符號。

2然後點選開啟「語言喜好設定」。

如果找不到右下角的語言符號，也可以從與下方式進入「語言喜好設定」:
開始 > 設定 > 時間與語言 > 語言

3點選新增語言，在跳出選擇語言的視窗後

4搜尋或是找到「日本語」，選擇並選下一步後點選安裝

完成!!現在妳可以透過 WIN + 空白鍵切換成日本語輸入法了

沒有特別調整的話預設會是羅馬拼音，試著輸入「ＮＩ　ＨＯ　ＮＮ　ＧＯ」打出日本語吧！

Windows 日本語輸入法的各種快捷鍵

此外，輸入法語言介面區塊下方的「上下鍵」，可以調整語言切換的順序，可以調整成自己習慣的方式。
另外，如果對英文輸入法沒有特殊需求，也可以移除英文輸入法，畢竟中文和日文輸入法都可以輸入英文，這樣可以讓切換語言不用三個語言間循環，但要注意某些軟體在不在英文輸入法快捷鍵會有一點點怪怪的。
