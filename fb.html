Marketing Research

使用族群定義

在上述分析後，我選擇以移動工作者為主要目標族群。為了在數據不足的狀況下定義目標使用者，我虛擬一個真實人物作為使用族群的縮影，再以其需求為近年筆電作簡單評比。
13吋以下小筆電（Tablet），以攜帶方便和性能不強為主要產品定位。除了功能不強之外，目前在市場上，產品定位與平板電腦重疊性太高，當然也不在行動工作族的需求範圍之內。而17吋以上筆電則是攜帶不方便。筆電大小在13~17範圍內，也有足夠空間達到的性能，所以在此範圍內一般CPU﹑GPU等規格，大多屬於偏價格問題，除非是類似於需要大量及時渲染的動畫製作，才超出這範圍，但是此類使用者通常使用桌機為主。
市場上，DELL XPS 13的超窄螢幕邊框以及MAC book的顯薄和質感，各種平板筆電式，皆是類似此類型，從市場可以看出除了性價比之外，窄邊、輕薄、質感、新的使用體驗皆為重點。
2015年筆記型電腦面板解析度FHD(1920x1080)以上的比例僅約18-19%。行動工作族通常產品規格需求會較高，且其中也不乏圖像創作和設計工作者，顯示器的解析度提高無疑可以劃分區隔，也較貼和使用者。

目標為15吋中高階筆記本，以及適合移動工作者的使用情境， 經過水平發展後，選擇將數字鍵盤改為觸控面板增加多元的互動方式，並在輕薄與筆電外觀上增添更多細節。

Product Description

Keyscreen - 數字鍵盤位置上的觸控式面板

15吋之上的筆電已有足夠的空間配置數字鍵盤。在行動使用情境之下觸控板反而是較重要的使用方式。在兩個基本前提下，將數字鍵盤改為觸控螢幕:Keyscreen。
Keyscreen除了支援原本的鍵盤功能之外，更能支援觸控板的使用情境，並且可以透過不同顯示模式也能讓交互使用觸控板﹑滑鼠時，擁有更加豐富的操作可能。

數字鍵盤 / 觸控板 / 自訂按鍵模式

Keyboard mode/Touchpad mode

數字鍵盤是keyscreen的預設模式。
除了原先的數字鍵盤，也可以安排自訂的按鍵或組合按鈕，也可以開啟滑鼠觸控面觸，此外控面板設於左邊能夠避免操作鍵盤時誤觸，也更貼近滑鼠使用模式。自訂鍵盤也能搭配應用程式，制定出特定的模組按鍵，如計算機等，創造出跟以往不同得使用體驗。

軟體介面模式

Softwarm mode

依照不同軟體特性的Keyscreen專門操作介面。
針對一般需求的基本配置會有撥放裝置﹑照片瀏覽。特別的如繪圖軟體的檢色器﹑剪輯軟體的時間軸﹑或是單純想是時間與氣象等等。
當使用Keyscreen時，某些應用螢幕會出現小視窗同步於顯示操作形況，可使眼睛不必盯著Keyscreen，操作上更為順暢。如繪圖的檢色器，使用時，也會同時出現在螢幕上。

視窗／繪畫模式

Screen/pan mode

選擇Keyscreen面板區塊同步顯示螢幕部分區塊。
選定區域內使用繪圖筆，進行小區域的簡單繪圖。適合短時間的使用行為，如：照片去背、筆記﹑草圖紀錄﹑臨時上色﹑簽名等等。在沒預計會長時間繪畫的情況下，可以不用特地帶繪圖板以及平板出門。
也可以搭派軟體的特殊應用，如：Painter的調色盤，此種方式可以不用另外開發螢幕應用，只要對準畫面即可使用。
